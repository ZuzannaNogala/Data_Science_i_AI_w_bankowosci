# Data Science and AI in Banking

This course was conducted in partnership with **Erste Bank Polska**. To successfully complete the course, three analytical projects were required:

### 1. Banking Sector Analysis
An analysis of monthly banking sector data for the period from December 31, 2009, to January 31, 2026. The data was sourced from the [Polish Financial Supervision Authority (KNF)](https://www.knf.gov.pl/?articleId=56224&p_id=18). 
* **Full Report (in Polish):** [`Nogala_list1`](https://github.com/ZuzannaNogala/Data_Science_i_AI_w_bankowosci/blob/main/list_1_solve.ipynb)

### 2. Telemarketing Campaign & Predictive Modeling
An analysis of telephone marketing campaigns conducted by a Portuguese banking institution from May 2008 to November 2010. This project was divided into two main parts:
* **Exploratory Data Analysis:** Investigating the campaign data. See the report (in Polish): [`Nogala_list2_p1`](https://github.com/ZuzannaNogala/Data_Science_i_AI_w_bankowosci/blob/main/list2_solve.ipynb).
* **Predictive Modeling:** Development and optimization of a classification model based on the XGBoost algorithm to predict a customer's propensity to open a term deposit. See the report (in Polish): [`Nogala_list2_p2`](https://github.com/ZuzannaNogala/Data_Science_i_AI_w_bankowosci/blob/main/list2_solve_3.ipynb).

### 3. Credit Card Customer Segmentation
An analysis of the “Credit Card Dataset for Clustering” (sourced from Kaggle), containing behavioral information on approximately 9,000 credit card users over a six-month period. The main objective of this project was to segment the customers based on their transaction habits.
* **Full Report (in Polish):** [`Nogala_list3`](https://github.com/ZuzannaNogala/Data_Science_i_AI_w_bankowosci/blob/main/list3.ipynb)
